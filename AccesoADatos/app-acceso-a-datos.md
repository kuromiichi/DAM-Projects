# Aplicación Acceso a datos

## Pantalla inicial de selección

Opción de elegir una categoría de un desplegable o editar las categorías en el almacenamiento elegido. 
Permite seleccionar entre local, SQL y Mongo.

### Editar categorías

Muestra una lista de categorías existentes. Permite seleccionar categorías y borrarlas con un botón, o crear una categoría nueva con otro botón.

## Vista de categorías

Muestra los elementos de la categoría elegida. Permite añadir elementos nuevos con un botón.
